Â¿Que es Git?

Git es un sistema de control de versiones. Un sistema de control de versiones es un registro de los cambios de un archivo o un conjunto de estos a lo largo del tiempo para poder recuperar versiones en un futuro.

Comandos Basicos:

1. Para inicializar un repositorio se hace con el comando git init

2. Para agregar los archivos utilizamos el comando git add

3. Luego para comentar un cambio usamos el comando git commit -m "cambio realizado"

4. Ahora para mandarlo a github usamos el comando git push y si es la primera vez que lo usamos tendremos que aclarar la ruta del repositorio
